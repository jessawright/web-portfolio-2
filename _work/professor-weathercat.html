Goal

Make a responsive single page app that displays current weather information for a given location and generates a Pusheen illustration for the weather.

Development

  • I used OpenWeatherMap's API for weather information. Their API includes easy-to-visualize categories of weather that I used to determine the elements of the Pusheen illustration.
  • I used the Google Maps Places Autocomplete API to allow users to search for a location.

Design

  • Many illustration elements are only very lightly modified from originals at Pusheen.com. Others were more heavily modified from originals (ex. combining multiple original illustrations into a single illustration element). I made the remaining illustration elements from scratch.
  • I designed the UI elements displaying the weather information to be sympathetic to Pusheen's illustration style and unobtrusive to the generated illustration. The font, Varela Round, imitates the lettering in original Pusheen illustrations. The transparency of the display boxes balances readability of the text with visibility of the illustration. Corners are rounded in keeping with the soft edges of Pusheen's world.
